# Vitrine de Projetos Aplicados SENAI

Plataforma digital para curadoria, organização e divulgação de projetos desenvolvidos por alunos da qualificação profissional do SENAI.

Este repositório reúne produções técnicas orientadas com foco em **aprendizagem aplicada**, **impacto social** e **resolução de problemas reais**, transformando trabalhos de curso em evidências concretas de competência.

---

## Propósito

Grande parte dos projetos desenvolvidos em cursos de qualificação acabam ficando restritos a entregas internas.

Esta iniciativa busca:

- 📌 Dar visibilidade aos projetos dos alunos  
- 📌 Valorizar a produção técnica e criativa  
- 📌 Conectar tecnologia com demandas reais da sociedade  
- 📌 Construir evidências pedagógicas sólidas  
- 📌 Fortalecer a formação profissional com propósito  

---

## O que você encontra aqui

- 📚 Projetos Front-End (HTML, CSS, JavaScript, React)
- ⚙️ Projetos Back-End (Java, Spring Boot, APIs REST)
- 🌐 Projetos de Redes (infraestrutura, Packet Tracer, servidores)
- 🧠 Projetos com impacto social (inclusão, acessibilidade, cidadania)
- 🏪 Soluções para pequenos negócios locais

---

## Metodologia aplicada

Os projetos são desenvolvidos a partir de uma orientação prática baseada em:

- Problemas reais (comércio local, serviços, sociedade)
- Construção de soluções digitais funcionais
- Integração entre front-end e back-end
- Evolução de ideia → protótipo → entrega
- Apresentação e comunicação (pitch)

---

## 🏆 Destaques

Os projetos em destaque são selecionados com base em:

- Qualidade técnica
- Clareza da proposta
- Aplicabilidade prática
- Organização do código
- Capacidade de entrega
